#Guide:
compile using the command:
gcc sat.cpp -o [output file name] -lglut -lGLU -lGL
run by typing ./[output file name]

Note: you need to have a cpp compiler like g++. You also need to have freeglut installed
if you dont have these preinstalled. run these commands in your terminal:
apt install g++
apt install freeglut3-dev
